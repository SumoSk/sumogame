Traffic Fixer Thailand - Playable Prototype v0.1
================================================

วิธีเล่น
1) เปิดไฟล์ index.html ด้วย Google Chrome / Microsoft Edge / Firefox
2) เริ่มเกมจะอยู่ในโหมดมือเปล่า คลิกถนน อาคาร Gate หรือรถเพื่อดู Inspector
3) เลือกเครื่องมือด้านซ้ายเพื่อวางถนนหรืออาคาร
4) ถนน: คลิกจุดเริ่ม แล้วคลิกจุดปลาย ระบบจะ Snap เป็น 90 องศาหรือ 45 องศา
5) อาคาร: คลิกวางบนพื้นที่ว่าง อาคารต้องติดถนนจริง ไม่งั้นวางไม่ได้
6) ลากเมาส์ขวา/กลางเพื่อเลื่อนแผนที่ ใช้ล้อเมาส์ซูม
7) ใช้ปุ่ม Pause / 1x / 2x / 5x / ข้ามวัน เพื่อควบคุมเวลา
8) เปิด Traffic Heatmap เพื่อดูถนนเขียว เหลือง ส้ม แดง ดำ
9) เปิด Debug Cell เพื่อดู occupancy, path และข้อมูลจำลองภายใน

ระบบที่ทำแล้ว
- แผนที่ 300x300
- เมืองไทย 4 เมือง: ภูเก็ต กรุงเทพฯ เชียงใหม่ โคราช
- Gate อยู่ขอบแมป แต่เริ่มเกมเป็นแผนที่เปล่า ผู้เล่นต้องลากถนนเชื่อมเอง
- Data กลางสำหรับถนน อาคาร บ้าน และ Profile ประชากร
- เมนูสร้างอ่านจาก Data กลาง
- ถนนเล็ก/กลาง/ใหญ่ พร้อมราคา ค่าบำรุง Capacity Speed
- วางถนน 90 องศาและ 45 องศา
- ทางแยกอัตโนมัติ และมีไฟแดงแบบสลับเฟสใน Prototype
- อาคารต้องติดถนนและตรวจพื้นที่ก่อนวาง
- บ้าน/งาน/ร้านอาหาร/ร้านค้า/ห้าง/โรงงาน/สวน/ผับ
- คนย้ายเข้าจาก Gate ไปยังบ้านว่าง
- Fixed Profile 100 แบบ ตามนิสัย ขยัน ชิล ๆ สายชิล บ้านรวย มนุษย์พิเศษ
- วันจันทร์-อาทิตย์มีผลต่อ Trip
- 1 คน = 1 รถ เมื่อเกิด Trip
- รถมีต้นทาง/ปลายทางจาก Citizen, Building หรือ Gate
- รถไม่ทับกันด้วย occupancy ต่อช่องถนน
- รถรอเมื่อช่องถัดไปไม่ว่างและรอไฟแดงเมื่อเข้าแยก
- Pre-Reservation สำหรับอาคารบริการ
- ถ้าไม่มีปลายทาง/ความจุเต็ม จะไม่สร้างรถและเกิด Demand Alert
- Pass/Fail Time Window และ Trip Fail จะ Reroute กลับบ้าน
- ระบบเงิน ภาษีประชากร ภาษีธุรกิจ ค่าบำรุงรายวัน
- Dashboard คำนวณจากสถานะจริง
- Inspector สำหรับถนน อาคาร Gate และรถ

ข้อจำกัด Prototype
- ยังไม่ใช่ production game engine เต็มรูปแบบ
- รถใช้ occupancy ตามจำนวนเลนในระดับ Tile แบบ Prototype ยังไม่ใช่ lane simulation ละเอียดระดับ production
- ไฟแดงเป็นระบบง่ายแบบสลับเฟส ไม่ใช่ระบบจราจรละเอียด
- อาคารโรงเรียน/โรงพยาบาล/ตำรวจ/สวนสนุกยังไม่ได้เพิ่ม เพราะ GDD ระบุเป็นระบบอนาคต
- กราฟิกเป็น 2D Canvas Prototype เน้นลื่นและทดสอบระบบก่อน

อัปเดต v0.2 Visual Upgrade
- ปรับการวาดถนนจากบล็อกสี่เหลี่ยมเป็นเส้นถนนต่อเนื่อง มีขอบถนน เงา เส้นแบ่งเลน และวงแยก
- ปรับรถจากวงกลมเป็นทรงรถ/รถบรรทุก มีทิศทาง กระจก ไฟหน้า ไฟท้าย และล้อเมื่อซูมเข้า
- ปรับ Gate ให้เด่นขึ้น พร้อมป้ายชื่อแบบอ่านง่าย
- ปรับอาคารให้มีมิติ เงา หลังคา ไอคอน และทางเข้าออกชัดขึ้น
- ปรับ UI เป็น Glass HUD สวยขึ้น อ่านง่ายขึ้น
- หมายเหตุ: ตั้งแต่ v0.5 เป็นต้นไป ถนนเริ่มต้นถูกปิดออก เพื่อให้เริ่มจากแผนที่เปล่า

อัปเดต v0.3 Road / Signal / Bot Trait Fix
- แก้ระบบถนน 2/4/8 เลนให้มีผลจริงกับจำนวนรถในช่องถนน ไม่ใช่แค่แสดงผล
  * ถนนเล็ก 2 เลน = รองรับรถใน Tile เดียวกันได้สูงสุด 2 คัน
  * ถนนกลาง 4 เลน = รองรับได้สูงสุด 4 คัน
  * ถนนใหญ่ 8 เลน = รองรับได้สูงสุด 8 คัน
- รถถูกวาดเยื้องตามเลนจริง ลดการซ้อนทับกันเมื่อวิ่งบนถนนใหญ่
- ปรับภาพถนนให้เห็นจำนวนเลนชัดขึ้น โดยถนน 4/8 เลนมีเส้นแบ่งเลนหลายเส้น
- แก้ระบบ Road Graph ไม่ให้ถนนที่แค่แตะมุมกันกลายเป็นทางเชื่อมปลอม
- การเลี้ยวเกิดจากทางแยกจริงที่ถนนตัด/ทับจุดเดียวกัน รถเลือกเลี้ยวเองตาม path ไม่ต้องสร้างถนนเชื่อมแปลก ๆ
- แก้ไฟแดงให้แต่ละแยกมีเฟสของตัวเอง ไม่สลับพร้อมกันทั้งเมือง
- ไฟแดงแยกเป็น 4 ทิศหลัก: east/west และ north/south พร้อม stop bar ในภาพ
- Inspector ถนนแสดงจำนวนรถบนช่องตามจำนวนเลน เช่น 3/8 คัน
- เพิ่มการกัน Event ตารางชีวิตซ้ำในวันเดียวกัน เพื่อให้ Bot รันตามอุปนิสัยไม่ซ้อนคำสั่ง
- Citizen แต่ละคนมี Fixed Profile ตั้งแต่เกิด/ย้ายเข้า และ Inspector รถจะแสดงอุปนิสัยของ Citizen
